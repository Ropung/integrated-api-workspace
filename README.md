## 프로세스 실행

- 전제 조건
    - [Docker Desktop]() 설치(또는 기타 수단으로 Docker Compose 수행 가능해야 함.)
- 실행: 루트 경로에서 수행
  ```shell
    docker-compose up -d
  ```

이제 각 서버가 실행되어 있다면, API 문서를 받아 올 수 있다.

접속 정보(Local): http://localhost:2999
